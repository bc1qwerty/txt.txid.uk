半減期 (Halving)
=============

2026-03-09 | ja

4年ごとにビットコインの新規発行量が半分に減少する。歴史上最も厳格な通貨政策。

半減期とは？
------

半減期(Halving)とは、ビットコインネットワークで約4年ごとに新規発行されるビットコインの量が半分に減少する事前プログラムされたイベントです。正確には210,000ブロックごとに1回発生し、1ブロックが約10分間隔で生成されるため、大約4年周期となります。

半減期はビットコインの通貨政策における最も重要な要素です。これは中央銀行の会議で決定されるのではなく、コードによって自動的に実行される不変のルールです。

半減期の技術的メカニズム
------------

ビットコインネットワークではマイナーが作業証明
[/ideas/proof-of-work/]を通じて新しいブロックを生成します。ブロックを正常に生成したマイナーは、ブロック報酬(block
reward)として新規発行されたビットコインを受け取ります。半減期はこのブロック報酬が半分に減少するイベントです。

## 半減期スケジュール

| 時点 | ブロック高 | ブロック報酬 | 日次新規発行量 (約) |
|---|---|---|---|
| 2009年 (誕生) | 0 | 50 BTC | 7,200 BTC |
| 2012年11月 (第1回) | 210,000 | 25 BTC | 3,600 BTC |
| 2016年7月 (第2回) | 420,000 | 12.5 BTC | 1,800 BTC |
| 2020年5月 (第3回) | 630,000 | 6.25 BTC | 900 BTC |
| 2024年4月 (第4回) | 840,000 | 3.125 BTC | 450 BTC |
| 2028年頃 (第5回) | 1,050,000 | 1.5625 BTC | 225 BTC |
| ... | ... | ... | ... |
| 約2140年 (最後) | 6,930,000 | 0 BTC | 0 BTC |

最終的には約2140年頃にブロック報酬が0になれば、合計2,100万個のビットコインがすべて発行された状態になります。その後はマイナーの収入が取引手数料のみで構成されます。

ストック対フロー比率 (Stock-to-Flow)
--------------------------

半減期の経済的意味を理解するために有用な概念がストック対フロー比率(Stock-to-Flow, S2F)です。

* ストック(Stock): 現在存在する総量
* フロー(Flow): 毎年新たに追加される量
* S2F比率: ストック ÷ フロー = 現在の生産速度で既存ストック量を置き換えるのに必要な年数

S2F比率が高いほど、資産はより希少であり、新規供給が既存ストックに与える影響(希釈効果)が小さくなります。

## 主要資産のS2F比較

* 金: S2F約60 (現在採掘された金の総量を年間採掘量で割ると約60年)
* 銀: S2F約22
* ビットコイン (2024年半減期後): S2F約120

2024年第4回半減期後、ビットコインのS2Fは金の約2倍に達しました。ビットコインは人類の歴史上最も高いS2Fを持つ資産となりました。そしてこの比率は次の半減期ごとに継続的に上昇します。

金採掘との比較
-------

ビットコインの半減期を理解するには、金採掘との比較が有用です。

金の供給構造:
* 金の年間採掘量は市場価格、採掘技術、地質学的発見により変動
* 金価格が上がると採掘インセンティブが増し、供給が増加 (部分的な自己調整)
* しかし正確な将来供給量を予測することは困難
* 歴史的に年間約1.5～2%の供給増加率

ビットコインの供給構造:
* 半減期スケジュールがコードで確定しており、将来供給が完全に予測可能
* 価格が上がっても総供給量は変わらない (採掘難度調整がこれを保証)
* 2024年以降の年間供給増加率は約0.8%で、金の半分以下

この違いは根本的なものです。金の供給は予測可能ですが確実ではありません。ビットコインの供給は数学的に確定されています。

法定通貨との対比
--------

半減期の意味は法定通貨 [/ideas/fiat-money/]の供給構造と対比するとき、最も明確になります。

法定通貨:
* 中央銀行が必要に応じて無限に発行可能
* 供給スケジュールが事前に定められていない
* 政治的判断により供給量が決定される
* 歴史的には、すべての法定通貨は長期的に価値が低下

ビットコイン:
* 誰も供給スケジュールを変更できない
* 今後100年の発行スケジュールが既に確定
* 数学とコードにより供給量が決定される
* 時間がたつにつれて新規供給が減少

法定通貨の通貨政策は「数名の専門家が会議室で決定すること」であり、ビットコインの通貨政策は「世界中の誰もが検証できるコードに刻まれたもの」です。

半減期と価格：相関関係と因果関係
----------------

歴史的にビットコイン価格は各半減期後に著しい上昇を示してきました：

* 第1回半減期 (2012): 半減期時点で約$12 → その後1年で$1,000以上に到達
* 第2回半減期 (2016): 半減期時点で約$650 → その後約18ヶ月で$20,000に到達
* 第3回半減期 (2020): 半減期時点で約$8,700 → その後約18ヶ月で$69,000に到達
* 第4回半減期 (2024): 半減期時点で約$64,000 → その後$100,000を突破

しかし、ここで重要な注意が必要です。

相関関係は因果関係ではありません。
半減期後に価格が上がったという事実が、「半減期のために価格が上がった」ことを証明するわけではありません。サンプル数が4回で、統計的に有意な結論を導くには少なすぎます。また、半減期はすべての参加者が事前に知っているイベントであるため、効率的市場仮説によれば、既に価格に反映されているはずです。

それでも供給面のロジックは明確です。他の条件が同じであれば、市場に毎日流入される新たなビットコインが半分に減れば、同じ需要を満たすために価格が上昇する必要があります。
これは基本的な需給の原理です。

予測可能な通貨政策の価値
------------

半減期の真の意義は、短期的な価格変動ではなく、予測可能な通貨政策という概念そのものにあります。

オーストリア経済学は、経済行為者が合理的な計画を立案するには、安定的で予測可能な通貨が必要であると強調しています。企業家が10年計画を立てるとき、通貨の供給がどのように変わるかわからなければ、すべての経済計算に不確実性が追加されます。

ビットコインの半減期スケジュールは、この不確実性を排除します。2030年、2050年、2100年にビットコインがどの程度発行されているかを、この瞬間に正確に知ることができます。これは人類史上どの通貨も提供できなかった水準の通貨政策の透明性です。

ここが時間選好
[/ideas/time-preference/]と結びつく地点です。将来の通貨供給を確実に知ることができるとき、人々はより長い時間軸で計画し、貯蓄することができます。予測可能な通貨政策は低い時間選好を可能にし、低い時間選好は資本蓄積と文明発展の基盤です。
